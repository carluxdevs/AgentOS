# 🔄 AgentOS Workflows (Standard Operating Procedures)
This folder contains the **blueprints** and **logic** for how agents interact within the POM.
## 🗝️ Core Principles
- **Loop-based Execution**: Return to a "Capital Allocator" (CPO) for validation.
- **Asynchronous Discovery**: Parallel research.
- **Strict Delegation**: Use of `<delegate_to>` tags.
