# 🚀 AgentOS: The Autonomous Product Operating Model (Skills Library)
AgentOS is an open-source framework that maps the topology of a modern tech company into an executable Multi-Agent System. This repository contains the **Intelligence Layer** (33 specialized AI roles and workflows) designed to be imported into any Agentic IDE or Framework.

## 🎯 Purpose: AI-Powered "Zero-Employee" Organizations
This is not a codebase for humans; it is a **Skill Library for AI Agents**. It follows the **Anthropic Pattern 1 (Claude)** to ensure maximum reliability and prevent hallucinations.
- 🧠 **33 Specialized Agents**: From CEO to DevOps, each with dedicated System Prompts.
- 🛠️ **Flat XML Tools**: Tool schemas designed for direct LLM function calling without nesting.
- 🧩 **Standard Workflows**: Defined interaction loops (see /workflows) for company orchestration.
## 🏗️ How to Use (IDE / Agentic Frameworks)
Since this is an Intelligence-only repository (The "Brain"), you can use it in two ways:
### 1. In AI-Native IDEs (Cursor, Windsurf, Replit)
Point your IDE to the `/skills` or `/workflows` folder. Your AI assistant will automatically learn the boundaries and logic of the Product Operating Model.
### 2. In Multi-Agent Frameworks (LangGraph, CrewAI, Autogen)
Import the `.md` files in `/skills` as your **System Messages** and use the blueprints in `/workflows` to design your graph logic.
